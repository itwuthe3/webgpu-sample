# Three.js サンプル

Three.js を使用したサンプルアプリケーション

## 概要

- React と TypeScript を使用してUIを構築
- Three.js を使用して3Dグラフィックスを描画

## 必要条件

- Node.js (バージョン 14 以上)
- npm 

## セットアップと実行手順

1. リポジトリをクローンします：
   ```
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```

2. 依存関係をインストールします：
   ```
   npm install
   ```

3. アプリケーションをビルドします：
   ```
   npm run build
   ```

4. サーバーを起動します：
   ```
   node server.js
   ```

5. ブラウザで `http://localhost:3000` にアクセスして、アプリケーションを表示します。
